# Weather Forecast Application

This is a responsive weather forecast application built using HTML, Tailwind CSS, and JavaScript.  
It fetches real-time weather data from the OpenWeatherMap API and displays both current weather conditions and a 5-day forecast.

## GitHub Repository
https://github.com/ramrohith999/weather_app
---

## Features

- We can search weather of a city by its name
- Get weather using current location
- Display temperature, humidity, wind speed, and description of the selected city
- 5-day forecast with daily weather details of the selected city
- Temperature unit toggle (°C / °F)
- Extreme temperature alerts including extreme cold and extreme heat
- Recently searched cities are stored using localStorage and are displayed in the dropdown even after refresh.
- Dynamic weather background patterns with symbols that match the weather
- Responsive design (Desktop, iPad Mini, iPhone SE)
- Proper UI-based error handling

---

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- OpenWeatherMap API
---

## How to Run

1. Open `script.js`
2. Replace the API key with your own OpenWeatherMap API key
3. Open `index.html` in your browser
